# 条件与循环

条件分合决流向，判明暗以择路；循环往复铸经纬，积跬步而至千里

## 一、条件语句

### 1、简述

条件语句，顾名思义就是编程中根据条件判断结果选择执行路径的语句，这在程序中十分重要，让程序得以选择在不同情况下如何运作，接下来我们先以Python为例，介绍最基础的if-else语句。

### 2、if-else

顾名思义，if代表的是“如果”，而else表达的是“剩下的”，直接刻画出了条件语句的不同分支。程序会先判断if后的条件是否为true（即变量中所讲布尔值中的true，或说1），再判断else if的，最后如果都不满足则执行else中的内容（没有则执行下文）。else if和else也可以不被使用，这样如果不满足if后的条件则直接执行下文，满足则先执行if中的内容再执行下文。以下是Python中的条件语句语法：

```
if 条件:
    命令1
elif 条件:
    命令2
else 条件3:
    命令3
```

其中的elif即else if（表示对“剩下的”进行下一步判断），另外需要说明的是，命令前的缩进（indentation）不可以省略，

## 二、程序循环
### 1、定义
程序循环定义如下（内容来自百度百科）：
> 程序循环是计算机程序中重复执行特定代码段的控制结构，通过次数设定、条件判断或集合遍历实现流程控制。其主要分类包括基于次数的循环、基于条件的循环（如while/until结构）以及遍历集合元素的循环，部分语言支持泛用循环结构。循环通常包含控制变量、初始条件、状态变化和终止条件四个要素。

程序循环的实现方式有很多，在此介绍最常见的while语句和for语句
### 2、while和for结构
在python中，while和for的表达式如下：

```
while 条件: #满足条件则一直循环
    命令
for 变量 in 范围:
    命令
```

值得注意的是，在python中for语句后面通常要接in来进行遍历操作，这一点和其他一些语言略有不同。比如在C中：
```
for(语句1 ; 语句2 ; 语句3){
    命令
}

```
这里的语句1一般用于初始化 ， 语句2用于判断条件 ， 语句3用于调整，类似整合版的while语句，常用于一定次数的循环，这样也可以使得只用于循环的变量的作用域（Scope）仅为循环结构的局部作用域中，详细内容不多赘述，感兴趣者可自行查阅相关资料

### 3、break与continue
这两个语句是用于跳出循环的，正如其名一般，break是指跳出当前整个循环，即“破坏”当前循环（嵌套则跳出其所在层），而continue则是指本次循环不再执行后文内容而是直接跳出，重新下一次循环，可与条件语句配合执行。

二者的语法在python中无特殊规范，直接输入该命令即可，在此不加示范
