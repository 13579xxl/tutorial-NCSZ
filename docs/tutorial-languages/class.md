# 一、定义
以下是类的定义（来自百度百科）：
>类（Class）是面向对象程序设计（OOP，Object-Oriented Programming）实现信息封装的基础。类是一种用户定义的引用数据类型，也称类类型。每个类包含数据说明和一组操作数据或传递消息的函数。类的实例称为对象。

这个定义看起来很深奥，实则讲述的内容很简单，接下来我们详细说说：

正如在函数章当中我们所说的，很多时候我们需要把一些东西抽象出来。不过类和函数又不大一样。在这里，学过物理的同学可能更好理解一点，在高中物理中，我们有一种方法叫做“整体法”，类和这个十分相似

举个简单的例子，比如说我想要做一个坦克大战的游戏，但是坦克有很多参数，还有移动等过程。存储数据，我们在前面讲过，可以使用变量，但是我们又没有“坦克”类型的变量，如果一个个定义参数，每个坦克都不一样，程序就会显得混乱不堪

有同学可能会说，那我们可以定义一个函数！很不错的想法，但是我们需要的操作和参数很多，而且我们不是同时调用这些过程，比如：我想要在按下某个键的时候发射子弹，按下另一个键则向前行走，这些过程是相互独立的

其实解决方法也很简单，我们要是能把很多组函数和需要的参数“捆绑”成一个“整体”，需要的时候再调用就好了。这就是“类”存在的意义

# 二、类的定义和使用
在python中，我们可以这样定义一个类，参考官方文档： https://docs.python.org/zh-cn/3/tutorial/classes.html）：
```
class ClassName:
    <语句-1>
    .
    .
    .
    <语句-N>```


```
其中的语句可以是变量的定义，也可以是函数，不过需要注意的是，函数的第一个参数表示类本身（通俗点说，就是给一个接口用于访问自身的数据），一般用self作为变量名（你当然也可以用其他的，但是习惯这种东西嘛，你懂的）。

另外，可以采用__init__函数来接受参数，即

```
def __init__(self , 参数):
    函数体 #一般是给内部变量赋值或者初始化
```

类的使用很简单：
```
#创建（即类的实例化）
var1 = ClassName(参数)
#调用内部函数
ClassName.function_one(参数)
```

非常易于使用
