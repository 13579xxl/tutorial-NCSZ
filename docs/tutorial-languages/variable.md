# 变量（variable）

## 一、定义
变量的定义如下（参考百度百科）：

> 变量来源于数学，是计算机语言中能储存计算结果或能表示值的抽象概念。变量可以通过变量名访问。在指令式语言中，变量通常是可变的；但在纯函数式语言（如Haskell）中，变量可能是不可变的。在一些语言中，变量可能被明确为是能表示可变状态、具有存储空间的抽象（如在Java和Visual Basic中）；但另外一些语言可能使用其它概念（如C的对象）来指称这种抽象，而不严格地定义“变量”的准确外延。 

简单来说，变量就像一个储存我们需要的数据的“小盒子”。在数学和物理中，遇到未知量，我们通常会设一个未知数进行求解，那么类似的，在程序中，我们则可以定义一个变量。

举个简单的例子，假设我们要制作一个比大小的程序：用户输入两个整数，程序自动返回较大的那个数字。不难发现，在真正实现这一内容的时候，由于我们不知道用户会输入什么，但我们需要对用户输入的数据进行处理，这时候我们就可以声明一个变量，即告诉计算机，我需要两个“小盒子”来存储两份数据，当用户输入数据后再将两个数据分别放入其中，当需要用的时候则可以直接从中取出。

:::caution

不可以随意使用空的变量，这就像想要从空盒子里拿出玫瑰花一样荒谬，你不是魔术师

:::

## 三、如何声明变量
在一些编程语言中，声明变量不需要给出变量类型，而再另一些语言中，则需要明确变量的类型。这两种方式并没有好坏之分，但是值得注意的是，即使不需要给出变量类型的语言，擅自对一种变量进行另一种变量的操作，也有可能引起报错，故而我们在声明一个变量时，一定要思考清楚我们需要的变量类型，以免产生bug

在此分别以C和Python为例，给出声明变量的方式


C语言：

`int var = 1 ; ` 

这里定义了一个名为var的整型（int）变量，并将其赋值为1，即将“1”这个数据储存到其中。

当然，也可以定义一个空变量，如：

`int var ;`

这就类似于告诉计算机你需要一个“小盒子”来存储数据，但是你现在并不打算使用它

Python:
 
 `var = 1` 

值得注意的是，在python中，我们并不需要给出变量的类型，而是直接给出变量名和值即可。
相似的，我们也可以定义一个空变量：

`var = None` 

这里赋值为None是Python中表示该变量没有任何值的方式，即“空”。值得注意的是，“空”和“0”有着本质上的差别，“0”本身是一个数据，和“1”、“2”、“3”并没有区别，而“空”则是代表没有数据 |

最后，值得一提的是，变量的赋值号为“=”，这个符号并不能表示等于，只能表示赋值。如果要判断两个变量是否相等，则需使用“==”符号
