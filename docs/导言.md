# 手册导言

## 一、简介

  该项目为简易教程，旨在让初学者可以快速上手进行小型项目的制作，从而激发学习者的兴趣，使其更好的进行更深入的学习和投入相关竞赛中

## 二、注意事项

- 除部分内容外，所有示例代码均使用 `python` 编写

- 所有内容均不具有权威性，有争议处以权威教材为准

- 本教程所有内容均以制作作品为目标，不可替代理论学习，仅供入门使用

## 三、其他

高中物理书中曾有过这么一段话，私以为用在此处也非常得当：

> “如果你对于物理学的博大精深及影响之巨仍然将信将疑，那也没有关系，还是先学下去吧，或许有一天你会拍打着教科书说："编者还欠费功夫。"

所谓计算机科学与电子信息科学，放眼整个人类科学史，它们出现的时间的确较晚，但是它们却和物理数学乃至化学生物等基础科学密不可分，故而真正要学好学透这些技术，就必须有深厚的自然科学功底。本教程目的在于快速上手，故而自然不会涉及较多物理学、数学知识，但是请不要因此而轻视这些技术，不要质疑它们的科学性和严谨性，当你继续学下去，当你真正掌握了这些技术底层的逻辑与原理之后，当你真正体悟到这些内容背后所蕴含的严谨与智慧之后，你也一定会反过头来笑着说，“编者还欠费工夫”。所以，请一直前进吧！用你的杰出作品来为这个充满数字的世界做出独属于你的贡献！

## 四、编者的话

邓小平同志强调：

> 计算机的普及要从娃娃抓起

诚然，随着智能手机的普及，很多同学有“操作电脑好像也不是那么重要”的思想。但是，计算机技术依然是现代社会不可缺少的一环。希望同学们能在阅读本册时有所感悟，有所收获。这便是我们编写本册的最终目的。
