# 资源推荐

# 写在前面

计算机这门学科是很共产主义的学科，你可以到处看见优质的免费视频教程，大把大把的开源项目可供学习，所以相较于“怎么学”，你更应该关注“学什么”，这样才能事半功倍，少走弯路。

## 一、编程基础相关

### 1.Bilibili资源
B站有数量相当的编程基础教学，而且是有视频讲解的，像是[黑马程序员](https://space.bilibili.com/37974444?spm_id_from=333.337.0.0)，他们有着系统，完善的cpp，python，java教程。至于其它语言，基本上，你需要做的事情是直接在搜索框搜对应的语言，然后依据以下优先级，逐个浏览，并判断是否值得学习

播放量>发布日期>弹幕量>时常

播放日期为什么如此重要，是因为越新的教程越有可参考性质，计算机是发展很快的学科，技术迭代快，旧的教程，很可能用的是老掉牙的技术，不值得学习。

时常为什么不重要，是因为一个七八十集的课程，其实很多有可能是废话，或者是无关紧要的知识点，一个十几集的教程，可能全是精华，所以这一点也需要分辨。

### 2.文本教程资源
文本的教程比较少，本人推荐[菜鸟教程](www.runoob.com)，这是一个比较全面的教学网站，但是实际上由于创作者很多，代码风格并不统一，所以笔者在此只推荐当作一个大纲，或者预习，来进行使用。

优质的内容一般在一些博客上可以看到，尤其是当你面对某个报错，最快的解决方法不是自己瞎调试，而是直接把错误信息输入到搜索框，看看能不能借鉴前人的经验，找到已有的解决办法尝试，这才聪明。
### 3.AI
不得不说，AI在编码能力上，是真的值得你信赖，所以笔者其实认为，如果你想以最高效的办法学习编程基础，就是先看视频，不懂得直接问AI，让他解释清楚。所以在此笔者推荐

- [Deepseek](deepseek.com)
deepseek的模型不差，但是容易触发限额，所以笔者再次推荐

- [通义千问](tongyi.com)
Qwen是阿里的模型，表现其实也挺好，只是也容易触发限额

以上是免费模型，其实如果你掌握VPN的使用，笔者强烈推荐尝试

- [ChatGPT](chat.gpt.com)
由OpenAI提供的GPT有着不俗的表现，前提是你得有一个账号，一个月大概会花你小两百块钱，但是对比之下，一套acwing的课程也差不多是一百多，还是没那么贵的。
- [Claude](claude.ai)
由Anthropic出品的claude模型在编程上很不错，只是你必须要一个账号，而且需要付费使用。

## 二、编程思维相关

###  1.游戏推荐
其实笔者本人还挺推荐寓教于乐这种思想的，所以很多时候，进行一些有利于编程思维的游戏，依笔者之拙见，也是对学习编程无害的。

首先是可以下到盗版的
- Human Resource Machine：这款游戏，其独特的设计，可以很好的培养流程控制思想。
- 7 billion humans：这款游戏和上面这个出自同一家工作室，不过难度更高，涉及到并发思维了。

其次是需要付费的游戏，可以在Steam上找到
- 编程农场：一个可以让你使用类似python语言的代码收割农作物的游戏
- 图灵完备：这游戏并简单，如果对计算机底层原理与电路逻辑设计的同学感兴趣的，可以尝试
